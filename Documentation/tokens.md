# Token Types
```javascript
// Single-character Tokens
left_paren("("), right_paren(")"), 
left_brace("["), right_brace("]"), 
comma(","), dot("."), 
minus("-"), plus("+"), 
semicolon(";"), slash("/"), star("*"), 

// One or two character tokens
bang("!"), bang_equal("!+"),
equal("="), equal_equal("=="), 
greater(">"), greater_equal(">="), 
less("<"), less_equal("<=")

// Literals
string, number, 
indentifiers(names of variables or functions)

// Keywords
&&(however we decide to identify and), else, false, funky(whatever word we decide to create functions), 
for, if, nish(whatever we decide to call null), or, print, return,
fact(whatever we decide to call variable), while
